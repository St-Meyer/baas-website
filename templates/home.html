BAAS Project Homepage

A simple debugging web interface

What is BAAS?

BAAS is a simple server-based solution to deploy images on dedicated machines. Such machines can be hardware-based, virtual or a combination. It works based on images which are combined into image setups which can be deployed to the server. This setup allows for a mix-and-match between various images to reduce duplication, ease experimentation and add extensibility to software imaging.

Why was it made?

The project was setup by professor Dr. J.S. Rellermeyer in order to support research at universities. It specifically targets OS and distributed computing tasks, but it can be used more broadly as well.

Who created it?

It is a joint student project of the Leibniz Universität Hannover and the Delft University of Technology. The backend was developed by Jonathan Dönszelmann, Victor Roest and Valentijn van de Beek. A webfront end for BAAS was created by Valentijn van de Beek with the help of Paulina.
